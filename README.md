## 📝 Features of the language

Grob emphasizes clarity and safety. The language uses static typing, which forces the system to check for errors before it runs your script. This process prevents many common mistakes that lead to crashes. The system also handles file operations as first-class activities. This means you can read, write, and organize files using simple commands rather than complex code structures. 

The underlying engine uses bytecode, a format that sits between human-readable text and machine code. This setup ensures that your scripts run at high speeds while staying portable. Because the system runs on .NET, it follows modern industry standards for memory management and performance.
