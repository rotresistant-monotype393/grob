# bool — Type Registry

## Methods

| Member | Signature | Notes |
|--------|-----------|-------|
| `toString()` | `→ string` | Returns `"true"` or `"false"` |

## Literals

```grob
true
false
```

## Examples

```grob
active := true
print(active.toString())   // "true"
```

See also: [Operators](../Language-Specification/Operators.md) for logical
operators `&&`, `||`, `!`.
